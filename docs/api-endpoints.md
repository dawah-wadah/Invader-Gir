# API endpoints

##HTML API

### Root
  -`GET /` - loads React web application

### Users
  - `GET/users/new`
  - `POST/users/`
  - `PATCH/users`


### Sessions
  - `GET/session/new`
  - `POST/session/`
  - `DELETE/session`

## JSON API

### Posts
- `GET/posts/`
- `POST/posts/`
- `PATCH/posts`
- `GET/posts/:id`
- `DELETE/posts/:id`

## Comments
- `GET/posts/:id/comments`
- `POST/posts/:id/comments`
- `PATCH/posts/:id/comments/:commentId`
- `GET/posts/:id/comments/:commentId`
- `DELETE/posts/:id/comments/:commentId`
